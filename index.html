Game Modes

Local Multiplayer
Play with a friend on the same device. Customize player names and enjoy real-time competitive gameplay with beautiful animations.

Play Computer
Face three difficulty levels: Easy (random moves), Medium (strategic blocking), or Hard (unbeatable Minimax algorithm).

Online Multiplayer
Play with friends remotely using shareable room codes. Real-time sync across browser tabs with built-in chat messaging.

Premium Features

🎨
4 Unique Themes
Classic, Neon Dark, Wooden, and Pastel. Each theme has its own color palette and vibe. Switch anytime without leaving the game.
🔊
Advanced Sound Design
Web Audio API synthesized sounds. Click effects, victory chimes, draw buzzes, and smooth transitions. Mute anytime with one click.
📊
Comprehensive Scoreboard
Track wins, losses, and draws for every player. Full game history with timestamps, modes, and results. Persistent storage via localStorage.
✨
Smooth Animations
Hand-crafted SVG symbols with stroke animations. Cell scaling on hover, winning cell flashes, screen fades, and board shake effects.
📱
Fully Responsive Design
Seamlessly adapts from mobile (375px) to desktop. Touch-friendly interface, optimized layouts, and perfect scaling on all devices.
♿
Accessibility Built-In
Full keyboard navigation with Tab and Enter. Clear focus indicators, semantic HTML, and screen reader compatible for all users.

Built With Modern Tech

🔧
Vanilla JavaScript
Zero dependencies. Pure ES6+ code with modular architecture. No frameworks, no bloat. Fast and lightweight.

localStorage & Events
Persistent game data. Real-time sync across tabs using storage events. Your stats and preferences are always saved.

Synthesized sounds without audio files. Full control over frequency, duration, and volume. Pure programmable audio.
